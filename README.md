# 🖼️ PCF Image Viewer Control

A custom PowerApps Component Framework (PCF) control that displays images from a URL with preview, delete functionality, and editable input.

## 🚀 Features
- Image preview from URL
- Delete button to remove image
- Input field to update image
- Responsive UI

## 🧰 Tech Stack
- PCF (PowerApps Component Framework)
- TypeScript
- Dataverse

## 📦 Setup

```bash
npm install
npm run build
pac pcf push --solution-unique-name YOUR_SOLUTION_NAME
